# MarathiEmoExplain
MarathiEmoExplain is a 10,762-sentence Marathi dataset annotated with sentiment, fine-grained emotion, and human-written justifications. It enables both classification and reasoning generation benchmarks, supporting interpretable, culturally grounded NLP research in a low-resource setting.
